# IRCTC 3-Statement Financial Model & Dashboard

**Live Portfolio Project | MBA Finance | Prepared by Kartik Rao**

## Project Overview
Built a fully integrated 3-statement financial model for IRCTC with 5-year forecasts across Ticketing, Catering, and Tourism segments. Developed an interactive Power BI + Streamlit dashboard with scenario analysis and sensitivity testing.

## Key Highlights
- Fully linked Income Statement, Balance Sheet & Cash Flow Statement
- Segment-wise revenue forecasting (Ticketing, Catering, Tourism)
- Scenario Analysis (Base / Bull / Bear cases)
- Sensitivity Analysis on growth rates and margins
- Interactive Dashboard with KPI cards and trend analysis

## Tools Used
- Microsoft Excel (Advanced Financial Modeling)
- Power BI (Dashboard & Visualization)
- Python + Streamlit (Live Interactive Dashboard)
